Electronic Paint Sensor

Prueba Sensor DIY Electronic Paint

Proceso.

  • Extracción del centro de lápices 9B, ya que tienen una mayor concentración de grafito.

  • Mezcla con pintura de acrilico y pintado sobre una hoja de papel.

  • Conexiones con protoboard y arduino.

  • Finalmente conseguimos el sensor con pintura conductiva.

Funcionamiento.

  • Tenemos el sensor construido.

  • Generamos el código.


// capacitive sensing includes
#include <CapacitiveSensor.h>

// capacitive sensing constant
CapacitiveSensor sensor = CapacitiveSensor(4,2);  // 1M resistor between pins 4 & 2, pin 2 is sensor pin

void setup() {
  Serial.begin(9600);
}

void loop() {
  long measurement =  sensor.capacitiveSensor(30);
  
  Serial.println(measurement);
  delay(10);
}

  • Vemos los resultados en el monitor serial al acercar nuestra mano.
  • Cuando vemos los valores caer significa que nuestra mano esta cerca del sensor, aun así debe ser una proximidad muy cercana para que lo logre detectar.
