Tutorial Hugo - Instalación + Implementación de Tema

Posted Mar 24, 2020

Video Tutorial

Paso 1: Descargas

Descargar archivos de Hugo: https://github.com/gohugoio/hugo/releases

Descargar tema personalizable: https://themes.gohugo.io/aether/

Paso 2:

  • Crear una carpeta llamada “Hugo” en el escritorio.
  • Dentro de la carpeta crear una nueva carpeta llamda “bin” y otra “sites”.
  • Abrir el .zip descargado de la pagina de Hugo.
  • Descomprimir los archivos de la descarga de Hugo Framework en la nueva carpeta “bin”. (El directorio debe ser C:\Users\Pablito\Desktop\Hugo\bin)

Paso 3:

  • Abrir CommandPrompt desde el buscador.
  • Usar el comando “cd C:\Users\Pablito\Desktop\Hugo\sites”.
  • Luego usar el comando “hugo new site “nombredelsitio”. Esto creará una nueva carpeta dentro de sites con el nombre que fue elegido.

Paso 4:

  • Abrir el archivo descargado del tema personalizado.
  • Descomprimir el archivo dentro de la carpeta “themes” que se encuetra en la nueva carpeta del sitio creada (C:\Users\Pablito\Desktop\Hugo\sites\nombredelsitio\themes)
  • Cambiar el nombre de la carpeta de “aether-master” a “aether”.
  • Abrir la carpeta “aether” e ir a la carpeta llamada “exampleSite”.
  • Copiar los archivos que se encuentran dentro de la carpeta “exampleSite” y pegarlos en “C:\Users\Pablito\Desktop\Hugo\sites\nombredelsitio”, aceptar reemplazar.

Paso 5:

  • Dentro del CommandPrompt digitar “hugo server”.
  • La pantalla debería mostrar el siguiente código:

Building sites … | EN ——————-+—– Pages | 35 Paginator pages | 7 Non-page files | 6 Static files | 75 Processed images | 13 Aliases | 14 Sitemaps | 1 Cleaned | 0

Built in 171 ms Watching for changes in C:\Users\Pablito\Desktop\Taller Integrado\Paginas Web\Hugo\sites\Implementacion_Tema{archetypes,content,data,layouts,static,themes} Watching for config changes in C:\Users\Pablito\Desktop\Taller Integrado\Paginas Web\Hugo\sites\Implementacion_Tema\config.toml Environment: “development” Serving pages from memory Running in Fast Render Mode. For full rebuilds on change: hugo server –disableFastRender Web Server is available at http://localhost:1313/ (bind address 127.0.0.1) Press Ctrl+C to stop

  • En la penúltima línea muestra “Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)”
  • El link a la página creada es “http://localhost:1313/”
  • Ingresar el link en tu buscador predeterminado y estará tu nueva página web.
